# QGIS Templates

## Print Layout Template

`print_layout.qpt` is a pre-designed QGIS print layout for producing publication-quality map exports.

### How to use

1. Open your styled QGIS project (generated by spatial-machines)
2. Go to **Project → Layout Manager → New from Template**
3. Select `print_layout.qpt`
4. The layout opens with:
   - Title block (auto-populated from project title)
   - Main map frame (linked to the map canvas)
   - Legend (auto-populated from visible layers)
   - Scale bar (linked to map frame)
   - North arrow
   - Attribution text (edit with your data source)
5. Edit the subtitle and attribution text
6. Export as PDF or PNG from **Layout → Export**

### Layout specs

- Page size: A3 Landscape (420mm x 297mm)
- Title: 18pt bold, centered
- Legend: right side, auto-updating
- Scale bar: bottom left, single box style
- North arrow: top right of map frame
- Attribution: 7pt italic, bottom right

### Customizing

- Resize elements by dragging handles
- Change fonts via item properties
- Add additional text frames, images, or tables
- Save your customized version as a new template
